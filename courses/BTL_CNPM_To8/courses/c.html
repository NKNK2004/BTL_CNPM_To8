Giới thiệu về C

Đây là bài học giới thiệu về ngôn ngữ lập trình C. C là một ngôn ngữ mạnh mẽ, được sử dụng rộng rãi trong phát triển hệ thống.

Đặc điểm chính:

  • Tốc độ nhanh
  • Kiểm soát bộ nhớ tốt

Biến trong C

Biến trong C được khai báo như sau:

int x = 10;

Biến là nơi lưu trữ dữ liệu trong chương trình.

Vòng lặp trong C

Vòng lặp for trong C:

for(int i = 0; i < 10; i++) {
    printf("%d", i);
}

Vòng lặp giúp lặp lại một đoạn mã nhiều lần.
